Windows Pyspark installation Guide

  1. Via terminal install pyspark using the following command

    pip install pyspark

  2. Download and unzip winutils from here

  3. Modify your environment variables in Windows and add to the path the bin containted in the downloaded folder

  4. In your python code you will need to add the following lines of codes

    from pyspark import SparkContext

import os

os.environ['HADOOP_HOME'] = "path to the downloaded folder"

sc = SparkContext.getOrCreate()
